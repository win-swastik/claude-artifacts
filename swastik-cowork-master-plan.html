Swastik's Cowork Master Plan

Cross-device architecture · Device assignment · Scheduled automation · Domain routing

1 — How Everything Stays In Sync

The core principle: Both Macs point to the same iCloud Drive folder as their Cowork workspace. Your memory (CLAUDE.md), all outputs, and reference files live there. Work done on your MacBook Air is immediately visible on your Mac Mini — you never redo context.

🟢 Mac Mini Rule

5 — Setup Sequence (Do This Once)

Create your shared Cowork folder in iCloud Drive
Make a folder called "Cowork" inside your iCloud Drive. This becomes the workspace folder you select in Cowork on both Macs. Everything syncs through here automatically.
iCloud Drive / Cowork /
├── CLAUDE.md (auto-created by memory skill)
├── memory/ (auto-created by memory skill)
└── outputs/ (your deliverables)
Set this folder as workspace on BOTH Macs
Open Cowork on your MacBook Air 16GB and select the iCloud Drive/Cowork folder. Do the same on your Mac Mini. Both machines now read from and write to the same location.
Run the memory setup on MacBook Air 16GB (once)
Use the productivity:memory-management skill to tell Cowork about you — your role, company, priorities, shorthand, team names, and key context. This writes CLAUDE.md to the shared folder. Both machines immediately benefit from it.
Set up all scheduled tasks on Mac Mini
Go to your Mac Mini, open Cowork, and use the schedule skill to create each automation from the calendar above. Because it points to the same iCloud folder, the outputs appear in the same place you work from on your MacBook Air.
Set up separate memory for Personal workspace
In Claude.ai (personal workspace / Max), create a Project for investments and one for personal matters. Projects in Claude.ai store context per-project — this keeps your investment thinking separate from your Winvesta work.
Verify and test one scheduled task
Set the daily briefing to fire in 5 minutes on Mac Mini, confirm it runs, and check that the output appears in your iCloud Drive/Cowork/outputs folder. If that works, the entire architecture is validated.
